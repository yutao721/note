
# 如何使用一套键鼠控制两台电脑

## 1. 介绍

### 1.1. 介绍

- 使用一套键鼠控制两台电脑，可以方便地实现多任务处理，提高工作效率。
- 本教程将介绍如何使用一套键鼠控制两台电脑，包括硬件准备、软件安装和配置方法。

### 1.2. 硬件准备

- 两台电脑，一台作为主控电脑，一台作为被控电脑。
- 一套键鼠，支持多设备连接的键鼠。

### 1.3. 软件安装

- 主控电脑和被控电脑都需要安装相应的软件，例如：Synergy、TeamViewer等。

## 2. 安装Synergy

### 2.1. 下载Synergy

- 在主控电脑和被控电脑上分别下载并安装Synergy软件。
- 下载地址：<https://symless.com/synergy>
- 选择适合自己操作系统的版本进行下载。
- 下载完成后，双击安装包进行安装。
- 安装过程中，选择默认选项即可。
- 安装完成后，打开Synergy软件。
- 在主控电脑上，选择“Server”选项，在“Server”选项卡中，点击“Start”按钮启动Synergy服务。
- 在被控电脑上，选择“Client”选项，在“Client”选项卡中，输入主控电脑的IP地址，点击“Connect”按钮连接到主控电脑。
- 连接成功后，就可以使用一套键鼠控制两台电脑了。
- 在主控电脑上，按下“Ctrl+Alt+方向键”组合键，就可以切换到被控电脑的屏幕。
- 在被控电脑上，按下“Ctrl+Alt+方向键”组合键，就可以切换回主控电脑的屏幕。
- 在主控电脑上，按下“Ctrl+Alt+Shift+方向键”组合键，就可以在两台电脑之间复制粘贴文本。

## 3. 安装TeamViewer

### 3.1. 下载TeamViewer

- 在主控电脑和被控电脑上分别下载并安装TeamViewer软件。
- 下载地址：<https://www.teamviewer.com>
- 选择适合自己操作系统的版本进行下载。
- 下载完成后，双击安装包进行安装。
- 安装过程中，选择默认选项即可。
- 安装完成后，打开TeamViewer软件。
- 在主控电脑上，选择“Host”选项，在“Host”选项卡中，点击“Start”按钮启动TeamViewer服务。
- 在被控电脑上，选择“Client”选项，在“Client”选项卡中，输入主控电脑的ID和密码，点击“Connect”按钮连接到主控电脑。
- 连接成功后，就可以使用一套键鼠控制两台电脑了。
- 在主控电脑上，按下“Ctrl+Alt+方向键”组合键，就可以切换到被控电脑的屏幕。
- 在被控电脑上，按下“Ctrl+Alt+方向键”组合键，就可以切换回主控电脑的屏幕。
- 在主控电脑上，按下“Ctrl+Alt+Shift+方向键”组合键，就可以在两台电脑之间复制粘贴文本。

## 4. 安装 `Mouse without Borders`

### 4.1. 下载 `Mouse without Borders`

- 在主控电脑和被控电脑上分别下载并安装 `Mouse without Borders` 软件。
- 下载地址：<https://www.microsoft.com/en-us/download/details.aspx?id=35460>
- 选择适合自己操作系统的版本进行下载。
- 下载完成后，双击安装包进行安装。
- 安装过程中，选择默认选项即可。
- 安装完成后，打开 `Mouse without Borders` 软件。

首先下载，在两台电脑上分别安装，打开。界面如下：
![1](https://fastly.jsdelivr.net/gh/yutao721/blogImage@main/img/20251209142243192.jpg)
在连接有键鼠的电脑上点NO，被控制的电脑上点击YES。

然后，点击NO的客户端会生成一串密码，同时显示该电脑的设备名。对应填入点击YES的电脑，点击LINK连接。

![2](https://fastly.jsdelivr.net/gh/yutao721/blogImage@main/img/20251209142243193.jpg)
之后连接成功，就可以是用这套键鼠控制两台电脑了。
![3](https://fastly.jsdelivr.net/gh/yutao721/blogImage@main/img/20251209142243194.jpg)

通过Mouse without Borders不仅可以实现键鼠共享，还可以共享剪贴板，跨设备拖拽复制文件。

> 注意：使用 `Mouse without Borders` 时，需要确保两台电脑在同一个局域网内。该软件只支持Windows操作系统。
>

## 5. 总结

- 使用一套键鼠控制两台电脑，可以提高工作效率，方便多任务处理。
- 本教程介绍了如何使用Synergy和TeamViewer软件实现键鼠共享，以及如何使用 `Mouse without Borders` 实现键鼠共享和文件拖拽复制。
- 在使用这些软件时，需要注意网络环境，确保两台电脑在同一个局域网内。
- 最好使用的是 `Mouse without Borders`，因为其功能更强大，使用更方便。
